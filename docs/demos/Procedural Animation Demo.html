01 / Entrance choreography

CSS keyframes, staggered by code. Each word animates in on its own delay — re-runs every time the slide opens. No video file.
02 / Generative field

Procedurally drawn.

A live canvas loop. Hundreds of points generated and connected by code at 60fps — a different pattern every render. Impossible as a static image.
03 / Data-driven

The numbers animate from the data.

Bars + counters built from an array. Change the data, the chart re-animates. The motion is a function of the values, not hand-drawn.
04 / Self-drawing line

The story is always there.

SVG path, drawn by code. The Storyline motif strokes itself on, then the line lands. Pure math — scales to any size, crisp forever.
05 / Ambient loop
Always moving.
A continuous JS animation loop. Brand chips orbit forever — ambient life on a section divider, generated and positioned by code.
