Color as surface · bold pairings

Any color can be the surface.

Decks lean on paper + navy, but the whole palette can carry a section as a bold color block. Each surface below lists the brand colors that pass WCAG as type on it — including cross-pair combinations (e.g. lavender on teal), now allowed wherever contrast clears.

Roles: Body = ratio ≥ 4.5, safe at any size. Large + Bold = ratio 3.0–4.5, for headlines (≥ 24px) / bold (≥ 19px) and UI/borders only — never body copy. Below 3.0 the pair is decorative-adjacent only (block beside block), never type on type. Raw ratios: see the Contrast matrix card.
